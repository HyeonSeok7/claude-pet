[1] 다운로드
─────────────────────────────────────────────────

아래 주소를 브라우저(Chrome, Edge, Whale 등) 주소창에 붙여넣고 엔터.
자동으로 ZIP 파일이 받아집니다 (약 108 MB).

https://github.com/HyeonSeok7/claude-pet/releases/latest

"Assets" 섹션에서 "ClaudePet-X.X.X-windows.zip" 클릭.


[2] 압축 풀기
─────────────────────────────────────────────────

1) 다운로드 폴더(기본: 내 PC → 다운로드)에서 방금 받은 ZIP 파일을 찾습니다.

2) ZIP 파일 우클릭 → "압축 풀기" 클릭 → "압축 풀기" 버튼.
   같은 폴더에 ClaudePet-X.X.X-windows 라는 이름으로 폴더가 생깁니다.

3) 그 폴더를 더블클릭해 열면 두 파일이 보입니다:
   • ClaudePet-X.X.X.msi   ← 설치 파일
   • 설치방법.txt           ← 간단 요약본 (이 파일은 상세판)


[3] 설치
─────────────────────────────────────────────────

1) ClaudePet-X.X.X.msi 를 더블클릭.

2) 만약 파란 화면에 "Windows 에서 PC를 보호했습니다" 라고 뜨면:
   → "추가 정보" 글자 클릭
   → 아래쪽에 나타나는 "실행" 버튼 클릭
   (서명 없는 개인 제작 앱이라 경고가 뜨는 것. 안전함)

3) 설치 마법사가 뜨면 Next → Install → Finish 순서로 클릭.
   특별한 옵션 건드릴 것 없음.

4) 설치 끝나면 시작 메뉴에 "ClaudePet" 아이콘이 생깁니다.


[4] 실행
─────────────────────────────────────────────────

1) 키보드 Windows 키 누르기 → "ClaudePet" 입력 → 엔터.
   또는 시작 메뉴에서 ClaudePet 아이콘 클릭.

2) 잠시 후 화면 우측 하단에 작은 병아리 캐릭터가 떠야 합니다.
   → 떴으면 끝!

3) 작업표시줄 시계 옆 ^ 아이콘 클릭하면 숨겨진 아이콘들이 보이는데,
   거기에 병아리 모양 트레이 아이콘이 있어요.
   우클릭하면 "펫 숨기기 / 밥 주기 / 종료" 메뉴가 나옵니다.


╔══════════════════════════════════════════════════════════════╗
║  문제 해결:  "Failed to launch JVM" 오류가 뜨는 경우          ║
╚══════════════════════════════════════════════════════════════╝

실행했을 때 빨간 X 아이콘과 함께 "Failed to launch JVM" 다이얼로그
만 뜨고 캐릭터가 안 나오면, Windows Defender(윈도우 기본 보안 프로그램)
가 앱 실행을 막고 있는 것입니다. 아래 순서대로 예외를 추가하면 해결됩니다.


[5] Windows Defender 예외 추가
─────────────────────────────────────────────────

1) 키보드 Windows 키 누르기 → "Windows 보안" 입력 → 엔터.
   (파란 방패 아이콘이 있는 앱)

2) 창이 열리면 왼쪽 목록에서 "바이러스 및 위협 방지" 클릭.

3) 화면 중간 "바이러스 및 위협 방지 설정" 항목 아래의
   "설정 관리" 라는 파란 글자 클릭.

4) 스크롤을 계속 아래로 내리면 "제외 항목" 섹션이 있습니다.
   그 아래 "제외 추가 또는 제거" 파란 글자 클릭.

5) 사용자 권한 묻는 UAC 창(노란 방패)이 뜨면 "예" 클릭.

6) "+ 제외 추가" 버튼 클릭 → 나타나는 목록에서 "폴더" 선택.

7) 폴더 선택 창에서 아래 경로를 따라 이동:
      내 PC → 로컬 디스크 (C:) → Program Files → ClaudePet
   ClaudePet 폴더를 한 번 클릭해서 선택한 뒤 "폴더 선택" 버튼.

8) 목록에 "C:\Program Files\ClaudePet" 이 추가됐으면 성공.


[6] 다시 실행
─────────────────────────────────────────────────

1) 시작 메뉴에서 ClaudePet 실행.

2) 캐릭터가 화면 우측 하단에 뜨면 완료!

3) 그래도 안 뜨면 한 번 재부팅 후 다시 시도.


╔══════════════════════════════════════════════════════════════╗
║  기본 조작                                                     ║
╚══════════════════════════════════════════════════════════════╝

• 좌클릭 : 캐릭터가 반응 (대사 말풍선)
• 더블클릭 : 점프
• 드래그 : 캐릭터를 원하는 위치로 이동 (위치는 자동 저장)
• 우클릭 : "처음부터 시작" 메뉴 (초기화)
• 트레이 아이콘 우클릭 : 숨기기 / 밥 주기 / 종료

• 키보드 타이핑만 해도 포만도 자동 회복 + 호감도 증가
• 시간 지나면 배고픔 → 트레이 "밥 주기" 로 포만도 채워주기
• 호감도 5레벨마다 대사 톤이 변해요


╔══════════════════════════════════════════════════════════════╗
║  제거 방법                                                     ║
╚══════════════════════════════════════════════════════════════╝

1) 시작 메뉴 → ClaudePet 아이콘 우클릭 → "제거" 클릭.
2) 또는 설정 → 앱 → 설치된 앱 → ClaudePet → 제거.
